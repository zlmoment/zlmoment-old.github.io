Project Ideas

1. HDFS Java API

Use Java API to perform various file operations to HDFS, such as to access, read, write and delete.

2. Calculating PI

Use MapReduce algorithm to calculate PI.

3. Matrix Multiplication

Use MapReduce algorithm to perform matrix multiplication for large matrices.

4. Different versions of WordCount

As Jimmy's book says, use different kind of implementations to perform WordCount and compare the performances. Different implementations could be normal way, in-mapper-combiner, combiner, etc.

5. Inveted Indexing

It is actually about matrix operation. Use MapReduce algorithm to scan over text and build the inverted index.

6. Page Rank

A parallel Page Rank implementation using MapReduce algorithms. Data will be provided.

7. LogProcess

Use MaoReduce to analyze the logs. This project needs custom data types.

8. Top 10 Password

Find the most used 10 password in a dataset.

9. Movie Data Analysis

Mainly use Pig Latin to perform the analysis.

10. New York Stock Ticker Data Analysis

Mainly use advanced Pig Latin scripts like UDF.

11. Baseball Statistics

Designed for using Hive. Find the player with highest runs for each year. link

12. Books Data Analysis

Also designed for using Hive. link

13. K-means

Running K-means (dataset Human Development Report: link): from Hadoop Cookbook. We can also use some tools to visualize the results as bonus.

14. Twitter Sentiment Analysis

Use Mahout to perform sentiment analysis on Twitter.

15. Crime Prediction

To predict crime based on historical data.

16. Prediction Demand for Taxis

Use historical data to predict demands for taxis.

17. Predicting Closed Questions on StackOverFlow

Use historical data to predict if a question on StackOverFlow will be closed. Designed for job chaining.
