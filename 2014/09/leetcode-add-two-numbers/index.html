Leetcode - Add Two Numbers

Add Two Numbers 解题报告。Github代码地址：LINK

题目

You are given two linked lists representing two non-negative numbers. 
The digits are stored in reverse order and each of their nodes contain 
a single digit. Add the two numbers and return it as a linked list.

Input: (2 -> 4 -> 3) + (5 -> 6 -> 4)
Output: 7 -> 0 -> 8

原题地址: LINK

解题思路

首先注意几点：

  • 逆序：链表存的整数是逆序，比如例子中 (2 -> 4 -> 3) 表示整数 342。输出的答案也是逆序。
  • 长度：两个链表的长度可能不一样，这个情况要格外处理。
  • 进位：两个对应的数位相加如果大于10，会向前进一位，这个情况要格外处理。
  • 最后，如果相加完还需要进位，要在结果上加1.

我们先来整理一下大体的思路，我们首先想到的可能就是对于l1和l2，分别从头开始扫描，并对相应数位上的数相加，如果大于10，则下次相加要进位1。当l1或l2之中有一个已经扫描完毕而另一个没有时，我们应该把另一个余下的数全部加到结果后面，同时也要注意进位的问题。最后，在此判断仍否需要进位，如果需要，则在结果最后加上1。框架的代码表示如下：
