Notes about Maven

Here is a brief introduction to Maven, and some basic commands.

Creating Project

Command:

mvn archetype:generate

In the interactive mode, here are some examples to show what they mean:

groupId: com.hackecho.hadoop
artifactId: WordCount
version: 1.0
package: com.hackecho.hadoop.example

You’ll get following project structure:

WordCount
|-- pom.xml
`-- src
    |-- main
    |   `-- java
    |       `-- com
    |           `-- hackecho
    |               `-- hadoop
    |                   `-- example
    |                       `-- App.java
    `-- test
        `-- java
            `-- com
                `-- hackecho
                    `-- hadoop
                        `-- example
                            `-- AppTest.java

or you can use un-interative mode directly:

mvn archetype:generate -DgroupId=com.hackecho.hadoop -DartifactId=WordCount -DinteractiveMode=false

Commands

Package:

mvn package

Compile:

mvn compile

Compile and test:

mvn test

Clean ./target:

mvn clean

Change project to Eclipse project:

mvn eclipse:eclipse

Undo previous command:

mvn eclipse:clean
